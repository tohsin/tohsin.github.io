Research and Projects

2D Localisation and SLAM

 

My implimentation of probabilistic robotics algorithms for localisationa and SLAM like Prticle Filter, Kalman Filter, EKF, and main implimentation of graph SLAM in a 2d pygame simulator

  • Backend optimises a pose graph of Gauss Netwon for optimisation using the schur trick

  • Frontend Impliments an Iterative closest point for the simulated point cloud and user has to guid robot to start point for loop closure

Visual Slam and Visual Slam Python

 

Implimentaion of several concepts in visual Slam like Visual Odometry, optical flow, bundle adjustment, pose graph, loop closure, Siding window and dense reconstruction using Gauss Newton or g2o for optimisation. Visual slam Python is meant to serve as python repo to prototype Visual slam concepts using python wrappers for g2o, pangolin etc.

Tomato harvesting Soft Robotic arm (Research Assistant)

Joined a team of 3 with a graduate student to design and impliment an almost cost effective soft robotic arm to be used for harvesting Tomatos in a farm.

  • Using a vaccum pump, Cassava and rubber as the mechanism to grab Tomatos and harvest them

  • Trained a YOLO algorithms to identify types of tamatoes as well as asses state of fruit (Ripe/ Unripe) and deploy a version on raspberry pI

  • Simulated manipulator forward and inverse kinematic motions using python robotics toolbox.

LOUISE -A Medical CHATBOT

Medical Cahtbot designed to suggest emergency suggestions and solutions by avoiding large datasets of intents required to train similar bots. Chat bot has a sperate SVC model trained to categorise halth problems based on symptoms and a seperate Transformer model to detect Symptoms as intents and using several simlarity algoritms, map the right symptoms. Spliting the backend like this avoids hourse of writing a lot of intents to map symptoms in phrases to diagnosis
